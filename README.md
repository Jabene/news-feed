## <ins>Overview</ins>

# News Feed
This app was a take home challange meant to utilize the NYT Top Stories API. 


### <ins>Setup</ins>
1. Fork and Clone the repo
2. Install library dependencies by running `npm install`.
3. Open React with `npm start`.
